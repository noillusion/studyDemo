## [存在重复元素](https://leetcode-cn.com/problems/contains-duplicate/)
给定一个整数数组，判断是否存在重复元素。

如果存在一值在数组中出现至少两次，函数返回 true 。如果数组中每个元素都不相同，则返回 false 。

示例 1：

>  输入：[1,2,3,1]
> 
>  输出：true

示例 2：

>  输入：[1,2,3,4]
> 
>  输出：false

示例 3：

>  输入：[1,1,1,3,3,4,3,2,4,2]
> 
>  输出：true

题解 1：Set之后比较数组长度。
```
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var containsDuplicate = function(nums) {
    return [...new Set(nums)].length < nums.length
};
```

题解 2：排序后遍历。
```
var containsDuplicate = function(nums) {
    nums.sort((a,b) => a - b);
    for(let i = 1; i < nums.length; i++) {
        if(nums[i] == nums[i-1]) {
            return true;
        }
    }
    return false;
};
```

题解 3：哈希表，空间换时间。
```
var containsDuplicate = function(nums) {
    let map = new Map();
    for(let i = 0; i < nums.length; i++) {
        if(map.has(nums[i])) {
            return true;
        } else {
            map.set(nums[i], 1);
        }
    }
    return false;
}; 
```
